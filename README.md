# Telecom-churn-prediction-Analysis-of-customers
This project focuses on preparing and analyzing telecom customer data to understand the factors that influence customer churn. The goal is to clean the dataset, engineer meaningful features, and perform exploratory data analysis (EDA) to extract useful business insights that can help in building predictive churn models.
